GitHub与本地建立连接

一、获取密钥

  1. 在git指令框中输入ssh-keygen

  2. 将显示的地址下的PUB文件用记事本打开并复制

  3. 在github的settings的SSH and GPG keys选项连接SSH keys

    二、连接Git与github

  4. 新建一个文件夹

  5. 在文件夹中右键单击打开Git输入框
    输入git init初始化仓库

  6. 在github中找到你的文件点击Clone复制地址

  7. git remote add origin https://……

  8. git pull origin master建立主分支

  9. git add index.html(文件名)

  10. git commit -m ‘说明’

  11. git push origin master即可将文件上传到github
