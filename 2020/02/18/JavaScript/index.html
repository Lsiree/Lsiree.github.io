一、数据类型

  1. JavaScript不区分整数和浮点数，统一用Number表示

  2. 不采用int、double定义变量，统一用var这种变量本身类型不固定的语言称之为动态语言

  3. 两种等值比较
    ==它会自动转换数据类型再比较，不准确，推荐使用===，不会转换数据类型

  4. NaN 表示Not a Number，当无法计算结果时用NaN表示

  5. Infinity 无限大，数值超过了JavaScript的Number所能表示的最大值

  6. 用 var 定义元素var a=2,b=3;
